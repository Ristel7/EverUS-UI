# everUS Frontend Design

The official visual interface for the everUS platform. 

## Tech Stack
* **Markup:** HTML5
* **Styling:** Vanilla CSS3 (Modular BEM Architecture)
* **Interactivity:** Vanilla JavaScript
* **Typography:** Poppins (Google Fonts)

## Features
* Premium SaaS visual identity
* Fully responsive layout (Desktop, Tablet, Mobile)
* Modular CSS component structure
* Glassmorphism & custom motion animations
